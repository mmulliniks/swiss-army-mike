01 \u00b7 Brand

Brand at a glance.

The one-page shorthand \u2014 what Kaleo is, what it feels like, and the ideas that anchor every decision on the site.

Name & meaning

Foundational

Kaleo (KA-LEH-OH) is a Greek word meaning "to invite." Every page, every headline, every CTA on this site is an invitation \u2014 gentle, warm, no-pressure. That single idea is what the whole visual language exists to reinforce.

Aesthetic direction

Look & feel

Modern crafted \u2014 confident, joyful, grown-up. The site is designed for parents, so it never reads as childish. Big serif headlines. Bold saturated brand colors. Emerald canvas that feels like a rich theater curtain. Marquee-lights across the top. Placeholder shapes and shadows lifted from theatrical poster design.

Personality words: joyful, vibrant, heart-felt, exciting, fun. But always with grown-up craft \u2014 no clip-art, no rainbow gradients, no cartoon vibes.

The "three don'ts"

Positioning

The site's positioning is anchored by three commitments repeated across pages:

  • No auditions \u2014 every child gets a place
  • No competitions \u2014 confidence over trophies
  • No costume fees \u2014 the price you see is the price you pay

These appear as a dedicated section on the Home page and echo through the program pages. Keep them prominent \u2014 they're the biggest competitive differentiator vs. dance / theater studios in the area.

Three button styles.

Uppercase, tracked, no rounded pills. Feels like theater ticket stock \u2014 confident and printable.

Primary

The only orange button on the page. Use for the most important action \u2014 usually "Enroll" or "Claim my trial."

color: #FAF3E4

Yes! I'm in

Gold (hover / success)

Appears as the primary button's hover state, and as the "form successfully submitted" confirmation.

background: #FBD223
color: #06201E

\u25c6 Button rules

Text style
Uppercase, 15px, 600 weight, 0.02em letter-spacing. Space Grotesk.
Padding
18px vertical, 30px horizontal.
Radius
4px \u2014 subtle, not pill-shaped. Feels like a ticket, not a rounded app button.
Icon
Right-arrow inline SVG, 16\u00d712px, 12px gap from text.
Hover
Primary \u2192 gold (with color flip). Secondary \u2192 cream border, slight bg tint. Both lift 2px on Y.
Frequency
One primary button per screen when possible \u2014 preserves its meaning.

06 \u00b7 Sections

Section anatomy.

Every page is composed from this pattern library. Each section explained: what it is, when to use it, and how to build it in Squarespace.

Hero

Top of every page

Anatomy: Eyebrow (mono, gold, uppercase) \u2192 huge italic-accent headline (DM Serif Display) \u2192 subhead (Newsreader italic) \u2192 CTA row (primary + secondary) \u2192 optional side pull-quote (Newsreader italic).

When: Every page opens with a hero. Program page heroes have a colored accent-line above the eyebrow, matching that program's color.

Squarespace: Fluid Engine section, 3-4 stacked text blocks + button block. Use Custom CSS to set the massive font-size via a hero-title class.

Ticker / marquee strip

Attention grabber

Anatomy: Yellow (or coral) full-width strip, mono uppercase text, diamond dividers, scrolls horizontally forever.

When: Under hero on Home. Optional on program pages for major announcements ("Now Casting: Fall Session").

Squarespace: Custom Code Block. Full snippet in section 09.

"Three don'ts" strip

Positioning

Anatomy: Big serif headline with a strike-through word ("we don't do"), three-column card row underneath with numbered "No. 01/02/03" labels.

When: Home page positioning statement. Also referenced in shortened form on program pages.

Squarespace: Text block for headline (use inline <s> tag) + three Summary Blocks or three-column layout with Text blocks.

Program row list

Directory-style

Anatomy: Full-width table-style rows, one program per row. Columns: number \u00b7 large italic program name \u00b7 short description \u00b7 grade+price \u00b7 circular arrow. Hover shifts row left/right.

When: Home "Five ways to join" and Classes "Current lineup." Each row's program name uses that program's accent color.

Squarespace: Custom Code Block or repeated Auto Layout with Summary Blocks; requires custom CSS for the border-row style.

Heart values (3-card)

Values / benefits

Anatomy: Centered headline + subhead, then three cards \u2014 each with a star icon (in its program color), serif title, editorial body.

When: Home page and About page value statement.

Squarespace: Native 3-column layout with Text + Image (SVG stars) blocks.

Testimonials (3-card)

Social proof

Anatomy: Section headline in serif italic, three tag-topped cards. Each card has a colored 3px top border (coral / gold / teal), italic Newsreader quote, mono uppercase attribution.

When: Every program page + Home.

Squarespace: Native Testimonials Section (7.1 has one) or three Text Blocks styled with Custom CSS to add the top border accent.

Enroll offer card

Pricing

Anatomy: Two-column layout. Left: mono price label \u2192 huge orange serif price ($140/mo) \u2192 note \u2192 italic headline. Right: bulleted "includes" list with gold dots + big primary CTA.

When: Every program page with pricing. Direct handoff to iClassPro for registration.

Squarespace: Two-column section with Text + Button blocks. Custom CSS on the price display and bullet dots.

Lead form + iClassPro card

Conversion

Anatomy: Two-column. Left: eyebrow + heading + brief note + iClassPro embed card (with the Class Schedules button linking to portal.iclasspro.com/kaleoarts/classes). Right: Form block with parent name, phone, and a program-specific field (grade / age).

When: Bottom of every program page + on the Enroll page.

Squarespace: Two-column. Left: Text + Code Block (paste iClassPro button HTML). Right: native Form Block with 3 fields.

FAQ accordion

Reference

Anatomy: Grouped by topic (Getting started / What to expect / Logistics / Cost). Each Q is a big serif line with a gold plus button that rotates to X on open. Answer expands below in italic Newsreader.

When: FAQ page only.

Squarespace: Native Accordion Block \u2014 does this out of the box. Just custom-color the plus icon to gold via CSS.

Final CTA

Bottom of every page

Anatomy: Centered, dramatic. Small "RSVP" eyebrow \u2192 giant italic "You're invited." serif headline (up to 220px) \u2192 subhead \u2192 primary + secondary CTAs. Ambient gold/orange gradient glow behind the text.

When: Every page ends with this exact block. Consistency = memorability.

Squarespace: Full-width section, Text Block + Button Block, custom CSS for the giant fluid font-size and radial-gradient background.

07 \u00b7 Imagery

What to shoot.

The site is designed to leave big spaces for real photography and video. Placeholder blocks throughout tell you exactly what content should go where.

Photography style

Direction

Tone: Candid, warm, mid-action. Show kids in the middle of doing \u2014 not posed group shots. Real light, real faces, real emotion. Slight warm color-grading works well with the emerald canvas.

Composition: Loose, generous negative space so headlines can overlap. Landscape 4:3 for photo blocks, 16:9 for hero moments, 1:1 for feature card portraits.

Avoid: Stock-looking overly-polished group photos, stage-managed smiles, overly-branded shirts that clash with the palette.

Photo placements

Where they go

Every program page has a 3-photo strip right after the hero: 2 photos + 1 video thumbnail in the middle, slightly rotated for a scrapbook feel.

  • Kaleo Kids: Kids on stage \u00b7 class highlight video \u00b7 rehearsal shot
  • Kazoo: Little ones in class \u00b7 Open Play video \u00b7 parent + child
  • Songwriters: Students at piano \u00b7 live performance video \u00b7 recording studio
  • Classes: Tap class \u00b7 choir club video \u00b7 guest artist
  • Spark Camp: Campers dancing \u00b7 showtime video \u00b7 final production

The About page needs a founder portrait of Renee (3:4 portrait ratio) plus a "team on final day" square photo.

Video

Motion

Formats: Vimeo or YouTube for embedded highlight reels. Native MP4 upload for short (under 10 sec) auto-playing motion loops if desired.

Length: Highlight reels 45-90 seconds. Never longer \u2014 parents scan.

Placeholder: Every 3-photo strip has one video slot in the middle with a gold play button overlay. Swap in a Vimeo/YouTube embed via Squarespace's Video Block.

Placeholder styling

Design detail

Until real imagery is captured, all photo slots use a subtle dashed border with a mono-uppercase gold tag showing what should go there. This makes the mockup a working spec \u2014 anyone can see exactly which content is missing.

08 \u00b7 Voice

How Kaleo talks.

The copy is already written \u2014 the tone is the reason it works. Match this voice when you add anything new.

Warm, direct, grown-up

Voice

Kaleo talks like a thoughtful friend who happens to run an arts school. Confident but never salesy. Warm but never syrupy. Concrete details ("one hour and forty-five minutes, one day a week") not marketing fluff. Never talks down to parents; never talks over kids.

Sentence patterns to lean into

Style
  • The specific over the general. "Five sessions a year, five live stage performances." Not "many performances throughout the year."
  • The negative frame. "No auditions. No competitions. No costume fees." Says what you're not before what you are.
  • The invitation. Every closing CTA is a warm invite, not a hard push. "You're invited. Come see what an invitation feels like."
  • Em-dash asides. Kaleo copy uses em-dashes for warmth: "\u2014 and most importantly, have fun." Don't strip these; they're the voice.

Never

Avoid
  • Exclamation points beyond one per page (marquee excepted)
  • Corporate hedging ("world-class," "cutting-edge," "state-of-the-art")
  • Cutesy voice aimed at kids on pages designed to convert parents
  • All-caps for anything except the mono meta-text
  • Emoji \u2014 the star and diamond icons already carry that role

10 \u00b7 Squarespace

Build guide, step by step.

The order to rebuild this in Squarespace 7.1 with Fluid Engine. Follow it sequentially; each step assumes the previous is done.

  1. Pick a Fluid Engine template

    Start with a template designed for creative businesses \u2014 Bogart, Palmer, or Alex are good starting points. Any 7.1 template will work; just make sure Fluid Engine is enabled (most new templates are).

  2. Set brand colors globally

    Go to Design \u2192 Site Styles \u2192 Colors and add all 8 brand hex codes plus the emerald bg and cream text as a custom palette. Squarespace lets you name each swatch.

    • Set the "Dark 1" theme background to #0D4A44 (emerald)
    • Set the text color to #FAF3E4 (cream)
    • Set primary button to #F65A2B (orange)

  3. Set brand fonts globally

    Go to Design \u2192 Fonts and load all four Google Fonts (they're all free). Assign:

    • Headings 1-3 \u2192 DM Serif Display, weight 400
    • Paragraphs \u2192 Space Grotesk, weight 400/500
    • Custom "Meta" style via CSS \u2192 JetBrains Mono
    • Custom "Quote" style via CSS \u2192 Newsreader italic

  4. Add the global Custom CSS

    Paste the global variables snippet from section 09 into Design \u2192 Custom CSS. Add the italic-gold-em snippet, the marquee lights CSS, and any other component snippets you'll use. This is your global stylesheet.

  5. Build the site navigation

    Squarespace nav: main links go to About, FAQ, Enroll (styled as a button). The Programs dropdown is a folder in the main nav containing links to all 5 program pages. Site title (top-left) links back to Home.

    • Enable "Cart / Enroll" button in the header
    • Style the button with primary CTA color via CSS

  6. Build the Home page section by section

    Add sections in order from the site map doc: Hero \u2192 Ticker (Code Block) \u2192 Three don'ts \u2192 Approach \u2192 Programs list \u2192 Heart values \u2192 Testimonials \u2192 Location \u2192 Final CTA. Use Fluid Engine to place each block precisely.

    • For each section, refer to the "Section anatomy" chapter for structure
    • Copy comes verbatim from the copy doc \u2014 do not rewrite it, it's carefully tuned

  7. Duplicate the Home template for the 5 program pages

    Once Home looks good, duplicate its structure for Kaleo Kids, Kazoo, Songwriters, Classes, Spark Camp. Swap copy per program, change hero accent color, and add the program-specific sections (Kaleo Kids has the "four classes" strip; Songwriters has albums; Spark Camp has the 5-day tracker).

  8. Wire in iClassPro on every program + Enroll page

    Paste the iClassPro button snippet (section 09) into a Code Block on each program page's lead-form section, and prominently on the Enroll page. Optionally, use iClassPro's fuller iframe embed if you want the class schedule to show inline instead of open in a new tab.

  9. Build FAQ with the native Accordion Block

    Use Squarespace's built-in Accordion Block. Add all 12 Q&A items grouped by topic (use Section headers between groups). Paste the FAQ Custom CSS from section 09 to color the plus button gold and match the fonts.

  10. Add photography and forms last

    Once all pages are structured, swap the placeholders for real photos (Squarespace's Image Block handles alt text and lazy-loading automatically). Set up your Form Blocks with the lead-form fields; connect them to your email or CRM through Squarespace's Storage settings.
